Was ist no Code?

Bevor wir in die tatsächliche Programmierung einsteigen, wollen wir einen Ausblick in die No-Code Umgebung wagen

wie der Name schon sagt wird keine bzw. kaum tatsächlicher Code in einer Programmiersprache geschrieben.

Die Programmierung erfolgt vielmehr über eine grafische Benutzeroberfläche, oft in einer Art Blockstil, bei der Drag n‘ Drop Artig Bausteine eingesetzt werden.

hier wurde mithilfe von Blocky ein Baustein geschrieben mit dem "Hello World" in die Console geschrieben wird

an der Seite ist der Zusammengesetzte Code in einer Programiersprache representiert, hierbei die Sprache Python
