Die Programmierung erfolgt vielmehr über eine grafische Benutzeroberfläche, oft in einer Art Blockstil, bei der Drag n' Drop Artig Bausteine eingesetzt werden.

Variablen und Arrays

Betrachten wir in Blocky die Leiste Variables, ist es möglich Variablen zu erstellen die im Code verwendet werden können.

Variablen können im Laufe des Codes verändert werden und Speichern verschiedene Arten von Daten.

In diesem Blocky beispiel sind die Variablen Namens: number , string und shoppingCart zu sehen.

Der Generierte Code ist in JavaScript, wie auch Python sind diese beiden Sprachen "loosely typed"

Eine andere beliebte Sprache ist Java, diese hat feste werte und hierbei wäre number vom Typen Int,

diese bestehen nur aus ganzen Zahlen, für Fließkommazahlen wäre ein float Datentyp der richtige.

Die Variable String währe hier vom Typen String.

Die Komlpexität der Variablen kann anhand von Listen oder auch sogennanten Arrays Steigen, diese können eine Liste von Variablen beinhalten.

Die Liste shoppingCart zeigt hierbei zwei Einträge mit Cheese und Tomato.

Die Variablen können anhand deren Namen im Verlaufe des Codes Aufgerufen werden.

Vertiefung Arrays und Objekte

In der Vorherigen Lektion wurden Strings als Variablen betietelt, jedoch sind diese Komplexer als primitive Variablen wie Integers und Booleans.

Strings sind eigentlich Objekte und bestehen aus eine Kette aus Chars, welche aus nur einem Character bestehen (z.B. "E")

Somit sind Strings keine primitiven Variablen sondern Objekte, Objekte können aus mehreren primitiven Variablen bestehen, jedoch wird dies Bereits komplexer als für diese Lektion Vorgesehen.

Betrachten wir die Definierten Arrays sind diese Eindimensional, und beinhalten viele Einträge unter einem Namen.

Möchte mann nur eine der gespeicherten Werte abrufen so nutzt man den Index, in dem Vorherigen Beispiel wäre im Index shoppingCart[2]

die Tomate hinterlegt da Blocky Arrays mit 1 beginnen. Zubeachten ist hierbei das Arrays gewöhnlich von 0 beginnen, in sprachen wie LUA beginnen diese jedoch mit 1.

Arrays können in der Entwicklung mehr Dimensional sein, desto mehr Dimensionen, desto Komplexer werden die Listen

Schleifen und Funktionen

Schleifen ermöglichen es mehrere Iterationen einer Ausführung zu wiederholen.

Sie werden sehr oft in Verbindung mit Schleifen verwendet um durch alle Positionen einer Schleife zu Iterieren.

Hier wurden alle Angaben im ShoppingCart mit den Zahlen 1 bis 3 ersetzt


Der Gesamte Block in Blocky könnte als eine Funktion Definiert werden. Funktionen können im Code Aufgerufen werden

und Beispielsweise auch innerhalb einer Schleife mehrfach ausgeführt werden.
